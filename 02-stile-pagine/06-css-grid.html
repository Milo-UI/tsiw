CSS Grid

CSS Grid è un sistema di layout a due dimensioni, ovvero permette di controllare righe e colonne insieme. Qui lo vediamo con esempi semplici, per capire quando usarlo e come sfruttarlo al meglio.

1) Cos'è CSS Grid (e come si differenzia da Flexbox)

CSS Grid è un sistema di layout a due dimensioni: ci permette di controllare contemporaneamente righe e colonne. Possiamo decidere la struttura del layout come fosse una tabella, ma con molta più flessibilità.

In breve:
Usiamo Grid quando ci interessa decidere dove stanno gli elementi usando righe e colonne, usiamo Flexbox quando ci interessa soprattutto l'allineamento in una sola direzione.

2) Il grid container: display: grid e grid-template-columns

Come per Flexbox, tutto parte dal container: aggiungendo display: grid i figli diretti diventano grid items. Poi definiamo quante colonne vogliamo e quanto spazio occupano.

Esempio: tre colonne fisse

1
2
3
4
5
6

Il container definisce la struttura con grid-template-columns: 120px 1fr 120px. Tutti i figli si assegnano automaticamente alle celle della griglia.

3) Tracce e unità: px, fr, auto, minmax()

Una "traccia" di Grid è una colonna o una riga. Possiamo usare varie unità per dire quanto è grande: px, %, fr, auto, minmax().

Regola pratica:
usiamo fr per dividere lo spazio in parti o frazioni, e minmax() + auto-fit per layout responsivi "che si aggiustano da soli".

2fr

La prima colonna è fissa, le altre due si dividono lo spazio: la terza prende il doppio dello spazio della seconda.

minmax(min, max)

Ogni colonna ha una larghezza minima di 120px, ma può crescere fino a riempire lo spazio grazie a 1fr.

4) Posizionare gli elementi con le linee (grid-column / grid-row)

Ogni griglia ha linee numerate: possiamo decidere da quale linea a quale linea si estende un elemento, oppure usare span per dire "prendi N colonne/righe".

header (col 1-3)
sidebar
main (col 1-4)
footer (col 1-4)

Qui il "main" usa grid-column: 1 / 4; quindi occupa dalla linea 1 alla 4 (3 colonne). Il footer usa 1 / -1 per prendere tutta la riga.
