CSS Grid

CSS Grid è un sistema di layout a due dimensioni: possiamo controllare righe e colonne insieme.

1) Cos'è CSS Grid

CSS Grid è un sistema di layout a due dimensioni: ci permette di controllare contemporaneamente righe e colonne. Possiamo decidere la struttura del layout come fosse una tabella, ma con molta flessibilità.

Usiamo Grid quando ci interessa dove stanno gli elementi su righe e colonne, usiamo Flex quando ci interessa l'allineamento in una sola direzione del contenuto di righe e colonne.

3) Tracce e unità: px, fr, auto, minmax()

Una "traccia" di Grid è una colonna o una riga. Possiamo usare varie unità per dire quanto è grande:
px, %, fr, auto, minmax().

Usiamo fr per dividere lo spazio in parti e monmax() + auto-fit per layout responsivi "che si aggiustano da soli".

Ogni griglia ha linee numerate: possiamo decidere da quale linea a quale linea si estende un elemento, oppure usare span per dire "prendi N colonne/righe".

6) Layout responsivi con auto-fit e minmax()

Un pattern molto usato è:
grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); per creare card che vanno a capo da sole quando lo schermo si restringe.

7) Quando usare Flexbox e quando usare Grid CSS

Regole pratiche veloci

  • Abbiamo una fila di elementi (o una colonna) e ci interessa solo lo spazio in quella direzione? -> Partiamo da Flexbox.
  • Stiamo progettando un layout di pagina (header, sidebar, main, footer) o una griglia di card con righe e colonne ben definite? -> partiamo da CSS Grid.
  • Se ci troviamo a pensare "questo elemento prende 2 colonne e 2 righe", è un problema da Grid.
  • Se il focus è allineare pochi elementi (es. logo, menu, pulsanti) su una riga, è un problema da Flexbox.

Takeaway
Spesso usiamo entrambi insieme: Grid per il layout principale, Flex per i componenti interni.
